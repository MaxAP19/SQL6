# SQL6
Creation of database indexes

## Описание:
- Проект посвящён изучению механизмов индексирования в PostgreSQL и их влияния на производительность SQL-запросов. В рамках заданий выполнялась оптимизация запросов и доказательство эффективности индексов с помощью EXPLAIN ANALYZE.

## Этапы выполнения проекта:
- Создание B-Tree индексов для внешних ключей и проверка их использования в запросах.
- Анализ работы индексов через EXPLAIN ANALYZE, управление планами выполнения.
- Создание функциональных, составных, уникальных и частично уникальных индексов.
- Использование Index Scan и Index Only Scan для оптимизации выборок.
- Подбор покрывающего индекса для снижения времени выполнения сложного запроса.

## Освоенные компетенции:
- Понимание принципов работы индексов и причин, по которым они могут не использоваться.
- Навыки анализа и оптимизации производительности SQL-запросов.
- Практика создания различных типов индексов под конкретные сценарии запросов.
- Работа с EXPLAIN ANALYZE для доказательства эффективности оптимизаций.
- Развитие инженерного подхода к улучшению SQL-кода и производительности БД.
